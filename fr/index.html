Qu'est-ce qu'Esprit Vif ?

Esprit Vif est une application gratuite de jeux mentaux conçue pour entraîner la mémoire, la logique et la concentration. Elle inclut Sudoku, Nonogrammes et d'autres défis cognitifs pour tous les âges.

Deux versions, la même expérience

Les deux versions incluent exactement les mêmes jeux et fonctionnalités. La version payante supprime les publicités pour soutenir le développement de l'application.
