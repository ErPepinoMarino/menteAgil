O que é Mente Afiada?

Mente Afiada é um app gratuito de jogos mentais projetado para treinar memória, lógica e concentração. Inclui Sudoku, Nonogramas e outros desafios cognitivos para todas as idades.

Duas versões, mesma experiência

Ambas as versões incluem exatamente os mesmos jogos e funcionalidades. A versão paga remove os anúncios para apoiar o desenvolvimento do app.
